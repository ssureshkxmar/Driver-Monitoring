# Mobile app settings

## Settings

- apiBaseUrl: API base URL.
- wsBaseUrl: WebSocket base URL.
- enableSpeechAlerts: toggle speech alerts.
- enableHapticAlerts: toggle haptic alerts.
- enableSessionLogging: toggle session logging.
- enableAutoCoordination: toggle auto coordination.

## Validation

The app validates API URLs for http/https and WebSocket URLs for ws/wss.
